Корень квадратный вычисляется одинарным нажатием после числа (базы):
число (база) => кнопка √

При попытке вычислить квадратный корень из отрицательного числа калькулятор выдаст ошибку.

Корень n-й степени из числа x набирать в следующей последовательности:
число (база) => кнопка n√x => степень => "равно"

При отрицательной базе и нечетной степени корня или при нулевой степени калькулятор выдаст ошибку.

N-ю степень числа x набирать в следующей последовательности:
число (база) => кнопка Xn => степень => "равно"

Изменение знака числа на противоположный набирать в следующей последовательности:
число (база) => кнопка "+/-"
